Key Findings from Model Shootout

✅ RF depth=10 wins

Deeper trees capture the spectral index interactions better than shallower ones with this dataset. max_depth=6 with min_samples_split=5 over-constrains the trees — it achieves the same recall but at lower AUC (0.728 vs 0.747). The imputer + scaler + RF pipeline is the right architecture.

⚠️ SMOTE & XGBoost need more data

SMOTE synthesises negative-space interpolations that don't represent real encroachment patterns at this tile resolution — it hurts Test AUC and recall. XGBoost's probability calibration expects more samples for scale_pos_weight to calibrate properly. Both would likely outperform RF with 200+ positive examples.

🔍 Val vs Test AUC inversion (XGBoost)

XGBoost scores 0.722 on val but only 0.648 on test — it over-fits to the 6 val positives. RF is more stable (0.543 val / 0.747 test) because balanced class weights distribute learning more evenly. With a small fixed val set, the RF's val AUC is an under-estimate, not an over-estimate.
