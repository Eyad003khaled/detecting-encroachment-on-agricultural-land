False Positive Reduction Journey
Post-Processing: Temporal Consistency Filter

For each tile, scores across all consecutive pairs (T1→T2, T2→T3, T3→T4) are evaluated together. Two cases:

# Case 1 — established-encroachment drift
IF n_pos_pos_pairs_above_thresh ≥ 1
   AND total_pairs_above_thresh ≥ 2 (MAJORITY_THRESH)
THEN dampen only pos→pos pairs × 0.6  # signal pair untouched

# Case 2 — all-negative tile drift
IF no_pos_pos_pairs AND n_neg_neg_above_thresh ≥ 2
THEN dampen all pairs × 0.6  # no signal pair to protect
✓ True encroachment tile
neg→pos pair fires high (signal)
pos→pos pairs dampened if anomalous
neg→pos score preserved → TP
⚠ Seasonal FP tile
≥2 neg→neg pairs all fire high
All scores dampened × 0.6
Score drops below thresh → TN

Case 1 preserves the neg→pos signal even when pos→pos windows drift — the neg→pos pair is never dampened, only the spurious pos→pos fires. This fixed a tile_01 FN where all 3 pairs fired, causing naive dampening to kill the true signal.

Key Findings (v5 clean model)

⚠ Circularity was masking true performance

The v4 model included pct_conv and pct_new — both derived from the same spectral rule used to generate labels. A single-feature baseline (pct_conv alone) achieves AUC 1.000 on the test set, proving the labels were perfectly predictable from one feature. Removing them dropped honest val AUC from 0.963 to 0.861. The 0.861 is the real number.

📊 SAVI and NDVI delta-std are the top signals

In the clean v5 model, SAVI_d_std and NDVI_d_std rank #1 and #2 (importance 0.061 and 0.058). Variability in vegetation index change — not mean change — best separates encroachment from seasonal fluctuation. MNDWI and NDWI also feature strongly, reflecting water-channel proximity patterns in the Nile Delta.

🌎 Geographic gap is the honest ceiling

Random-split val AUC 0.861 is optimistic because nearby sites land in both train and test. The geographic holdout (train south, test north and vice versa) gives mean AUC 0.702 — the realistic floor for deployment in unseen areas. Bootstrap 95% CI [0.74, 0.96] shows the point estimate uncertainty on 45 val samples.
